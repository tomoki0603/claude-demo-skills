# nikkei-summaries — 日経投資ブリーフィング 出力フォルダ

## 概要

`nikkei-investment-briefing` スキルが生成した日経電子版の投資ニュースブリーフィングを格納するフォルダ。

---

## フォルダ構成

```
nikkei-summaries/
├── README.md                          ← このファイル
├── 2026-03/
│   ├── nikkei-summary-2026-03-08.md   ← 完全版（朝刊＋夕刊）
│   ├── nikkei-summary-2026-03-08-morning.md  ← 朝刊のみ（夕刊取得前の一時ファイル）
│   ├── nikkei-summary-2026-03-07.md
│   └── ...
└── 2026-04/
    └── ...
```

### ファイル命名規則

| ファイル名パターン | 説明 |
|-----------------|------|
| `nikkei-summary-YYYY-MM-DD.md` | **完全版**（朝刊＋夕刊の統合ファイル）|
| `nikkei-summary-YYYY-MM-DD-morning.md` | **朝刊のみ**（夕刊実行後に自動削除される一時ファイル）|

---

## スキルの実行方法

Claude Codeのチャットで以下のいずれかを入力:

```
「日経をまとめて」
「今日の株式ニュースをまとめて」
「朝刊をチェックして」
「夕刊を追加して」
「日経電子版を確認して」
```

---

## 朝刊・夕刊の運用フロー

```
【朝】スキル実行
  ↓
完全版ファイルがない場合 → 朝刊取得・保存
  nikkei-summary-YYYY-MM-DD-morning.md を生成

【夕】スキル再実行
  ↓
朝刊ファイルが存在する場合 → 夕刊を追加して完全版を生成
  nikkei-summary-YYYY-MM-DD.md を生成
  nikkei-summary-YYYY-MM-DD-morning.md を削除

【2回目以降】スキル実行
  ↓
完全版ファイルが既に存在 → 「本日分は生成済みです」と通知してスキル終了
```

---

## ブリーフィングの構成

各ブリーフィングファイルは以下の構成で出力される:

```
# 日経電子版 投資ブリーフィング — YYYY/MM/DD（朝刊）

## 🏦 金融政策・マーケット
### 記事タイトル
[記事概要・投資インパクト説明]
| 投資インパクト | ★★★ |
| 🔗 元記事 | [読む](URL) |

📖 **この記事の用語解説**
**FRB**: [解説]
...

## 📈 個別銘柄・業界
...

## 🌍 地政学リスク
...

## 📊 経済指標
...

## 💡 テクノロジー・産業トレンド
...
```

---

## 前提条件

- **Playwright MCP**: `.mcp.json` に設定済みであること
- **ログインセッション**: 日経電子版のログイン済みプロファイルが `C:/Users/start/.playwright-nikkei-profile` に保存されていること
- **Claude Code再起動**: `.mcp.json` を初めて設定した場合は要再起動

---

## トラブルシューティング

問題が発生した場合は `.claude/skills/nikkei-investment-briefing/` のスキルファイルを確認するか、
`skill-creator-max` スキルでスキルの改善を行う。
